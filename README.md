# GroovePlay 🎵

A premium music distribution and streaming platform built with modern Android technologies.

## 🚀 Recent Improvements

### 🎨 UI & UX Overhaul
- **Premium Entrance**: Integrated the official Android 12+ `SplashScreen` API with custom **Animated Vector Drawables (AVD)** for a high-end, animated launch.
- **Modern Home Screen**: A full Material 3 redesign featuring personalized headers, horizontal playlist carousels, and modern track cards.
- **Unified Branding**: New professional **Vinyl Icon** and geometric wordmark, also utilized as the official **Adaptive Launcher Icon**.
- **Dark & Light Mode**: Comprehensive support for system-wide themes with high-contrast, professional color palettes.

### 🏗️ Architecture
- **Jetpack Compose**: 100% declarative UI for a fast, responsive experience.
- **MVVM Pattern**: Clean separation of concerns using ViewModels and StateFlow.
- **DataStore**: Persistent theme preferences management.
- **Navigation**: Type-safe navigation graph using standard Jetpack Navigation.

## 🛠️ Technical Stack
- **UI**: Jetpack Compose (Material 3)
- **Icons**: Material Icons Extended
- **Splash**: AndroidX Core SplashScreen API
- **State Management**: Kotlin Coroutines & Flow
- **Dependency Management**: Gradle Version Catalogs (TOML)
